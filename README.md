# 📒 دفترچه هزینه

اپ دفترچه ثبت هزینه‌های روزانه — PWA با تقویم شمسی

## 🚀 دیپلوی روی Cloudflare Pages

۱. این مخزن رو روی گیت‌هاب Fork کن  
۲. برو به Cloudflare Pages → Create project → Connect to Git  
۳. مخزن رو انتخاب کن → Deploy  

## 📲 نصب روی گوشی

۱. URL Cloudflare رو تو Chrome باز کن  
۲. صبر کن تا گزینه Add to Home Screen بیاد  
۳. اپ مثل یه برنامه عادی نصب میشه  

## ⚙️ امکانات

- 📅 تقویم شمسی کامل  
- 🔍 فیلتر تاریخ، جستجو و مرتب‌سازی  
- 💾 بکاپ JSON / Excel  
- ✅ انتخاب گروهی و حذف  
- 🟢 قابلیت نصب PWA
