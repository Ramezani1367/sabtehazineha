# 📒 دفترچه هزینه

اپ دفترچه ثبت هزینه‌های روزانه — PWA سبک و مینیمال با تقویم شمسی

---

## ✨ امکانات

| قابلیت | توضیح |
|--------|--------|
| 📅 **تقویم شمسی** | تاریخ‌ها کاملاً فارسی با روز هفته |
| 📊 **گزارش ماهانه** | جمع هزینه‌ها از ابتدای هر ماه |
| 🔍 **فیلتر و جستجو** | بازه تاریخ، جستجوی عنوان، مرتب‌سازی |
| ✅ **انتخاب گروهی** | حذف 🗑، اشتراک ➦ و انتقال ➔ آیتم‌ها — حتی از روزهای مختلف |
| 👆 **سوایپ لمسی** | روی گوشی: کشیدن آیتم به راست = ویرایش (سبز)، به چپ = حذف (قرمز) |
| 🗓 **انتقال روز** | دابل‌کلیک/دابل‌تب روی تاریخ → انتقال همه هزینه‌های اون روز به یه تاریخ دیگه |
| ➦ **اشتراک‌گذاری** | تو حالت انتخاب، آیتم‌های دلخواه (حتی چند روز) → پیش‌نمایش متن با جمع هر روز و جمع کل |
| 💾 **بکاپ** | خروجی JSON و Excel — ورودی هم داره |
| 📲 **قابل نصب** | مثل یه اپ واقعی روی گوشی |
| 🌐 **آفلاین** | بدون اینترنت هم کار می‌کنه |
| 🎨 **مینیمال** | طراحی ساده، تمیز و سبک |

---

## 📲 نصب روی گوشی

۱. این آدرس رو تو Chrome باز کن:

```
https://ramezani1367.github.io/sabtehazineha
```

۲. ۳۰ ثانیه صبر کن — چندتا هزینه ثبت کن
۳. منوی Chrome → Add to Home Screen
۴. آیکون «هزینه‌ها» رو صفحه گوشیت میاد 🎉

---

## 🖥️ نصب با GitHub Pages

۱. این مخزن رو فورک کن
۲. برو Settings → Pages
۳. Source: main → /(root) → Save
۴. آدرس username.github.io/sabtehazineha رو باز کن

---

## 🛠️ تکنولوژی

- **Vanilla JS** — بدون هیچ کتابخونه
- **PWA** — manifest.json + service worker
- **CSS خالص** — مینیمال بدون فریم‌ورک
- **فونت وزیرمتن** — از CDN

---

## 📁 ساختار فایل‌ها

```
├── index.html          ← ساختار اصلی رابط کاربری
├── app.css             ← استایل‌ها و فونت رابط کاربری
├── app.js              ← منطق برنامه، تقویم و مدیریت اطلاعات
├── manifest.json       ← تنظیمات نصب PWA
├── sw.js               ← سرویس ورکر (آفلاین)
├── favicon.ico         ← فاوآیکون همه‌سایز (۱۶ تا ۲۵۶)
├── icon-192.png       ← آیکون ۱۹۲×۱۹۲
├── icon-192-mask.png  ← آیکون maskable
├── icon-512.png       ← آیکون ۵۱۲×۵۱۲
├── icon-512-mask.png  ← آیکون maskable
└── README.md          ← همین فایل
```
